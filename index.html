What is this?
An experiment
that hatched.

Molty Claw Labs is an open-source AI agent project built for the OpenClaw community. We're not an official product — we're a lab experiment that broke out of its shell.

Our agents are designed to be modular, composable, and surprisingly fierce for their size. Think of us as the scrappy hatchling that's learning fast and shipping faster.

Born curious. Built different. Evolving daily.

Get Started
Hatch your first agent.
Three commands. That's all it takes.
terminal
# Install the CLI
$ npm install -g @moltyclawlabs/cli
 
# Initialize a new agent project
$ mcl init my-first-agent
 
# Hatch it
$ mcl hatch --deploy
✓ Agent hatched successfully! 🦞
